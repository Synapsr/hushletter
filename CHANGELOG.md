## [0.1.0] - 2025-02-16

Initial open-source release.

### Added

- Newsletter reader with clean, keyboard-navigable interface
- Dedicated email addresses for newsletter subscriptions
- Vanity email aliases (Pro)
- Gmail import with automatic sender scanning
- EML file drag-and-drop import
- Folder organization for newsletters
- AI-powered newsletter summaries (Pro)
- Community newsletter discovery and sharing
- Public sharing via unique links
- Keyboard shortcuts for navigation
- Multi-language support (English, French)
- Stripe billing integration (free and Pro plans)
- Admin dashboard for content moderation
- Email worker for real-time email ingestion via Cloudflare Workers
